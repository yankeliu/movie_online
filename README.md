# node + express + mongoDB + gulp 搭建在线观影网站
## 功能说明
- 支持管理员、普通用户登录
- 管理员上传、修改、删除电影数据，
- 普通用户搜索电影、观看电影、评论等功能

## 运行项目

\#克隆项目到本地

`$ git clone git@github.com:yankeliu/movie_online.git`

\#切换到项目所在文件夹

`$ cd movie_online`

\#安装依赖

`$ bower install`

`$ npm install`

\#运行

`gulp`

*说明：运行之前需在本地安装 mongoDB 并启动该数据库*
